## Step 2: Set up your data file

The tool reads your personal details from a file called `my_data.csv`. A template is included in the download.

1. Find `my_data.csv.example` in the same folder as the app.
2. Make a copy of it and rename the copy `my_data.csv`.
3. Open `my_data.csv`:
   - **Windows**: right-click → Open with → Excel (or Google Sheets)
   - **Mac**: double-click (opens in Numbers), or right-click → Open with → Excel
4. Fill in the **Your data** column with your information. The **Notes** column explains each field.
5. Save the file.

For a full explanation of every field, see [my_data.csv reference](my_data_reference.md).

> `my_data.csv` stays on your computer — it is never uploaded anywhere.
